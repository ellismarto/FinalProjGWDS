CARP Posters

Below are my 3 final Bristol Balloon Fiesta CARP posters

For the first poster I went with a more basic and clean style. I cropped an existing image of the balloon fiesta in order to create space for the type and logos, and to be able to properly align and compose all of the elements. For the contrast element I made sure to show minimal balloon's so it didnt seem crowded, but at the same time have enough colour from the baloon's showing through. Everything is aligned across the left side of the poster, to make sure it is easy to read and not too overcrowded and confusing. As well as this it shows repitition. For proximity I made sure everything was spaced nicely and sized correctly, I seperated the logo, type and sponsers.
